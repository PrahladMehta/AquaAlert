# AquaAlert

Drink Water Reminder Extension

Chrome extension that:

- reminds you to drink water every minute for testing
- shows an animated water-drop reminder window
- lets you confirm each drink and adds 500 ml
- tracks today's total plus weekly and monthly history

## Install

1. Open Chrome or Edge.
2. Go to `chrome://extensions/`.
3. Turn on Developer mode.
4. Click `Load unpacked`.
5. Select this folder: `C:\Users\Prahlad\Projects\drink-water`

## Use

- Keep the extension installed and enabled.
- Every minute, a reminder window opens while testing.
- Click `I Drank Water` to add 500 ml.
- Click the extension icon to open the dashboard.
- Water can only be logged from the popup button or the reminder window.
